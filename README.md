## Deskripsi
Praktikum ini bertujuan untuk mempelajari pembuatan tampilan
antarmuka (UI) modern pada aplikasi Android menggunakan
**Jetpack Compose** dan **Material Design 3**.

- MaterialTheme
- Typography
- Color Scheme
